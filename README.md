# Customer Churn Prediction (AI Project)

This project uses a machine learning model to predict customer churn based on usage and billing data.

## Tools Used
- Python
- Pandas
- Scikit-learn

## What I Did
- Cleaned and prepared real-world dataset
- Built a Random Forest model
- Predicted customer churn
- Evaluated model performance

## Why This Matters
Customer churn prediction helps businesses retain customers and increase revenue through data-driven decision making.
